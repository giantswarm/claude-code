## Labels

**Required:**
- `team/<name>` - Delivery owner
- `kind/epic`

**Recommended:**
- `area/kaas` or `area/platform`
- `function/<domain>` - For cross-functional epics
- `working-group/<name>` - If applicable
- `start-date` - YYYY-MM-DD
- `target-date` - YYYY-MM-DD
- `quarter` - Q1 2026

## Examples

Good epic titles:
- `[EPIC] Production-Ready KubeEdge Solution for WEPA Edge Device Integration`
- `[EPIC]: Edge to Cloud Container Platform Campaign`
- `[EPIC] Eliminate service continuity risk by migrating away from discontinued OpsGenie`

## Quality Checklist

- [ ] Clear strategic motivation
- [ ] Explicit scope boundaries (in/out)
- [ ] Measurable outcomes defined
- [ ] Acceptance criteria are verifiable
- [ ] Dependencies identified
- [ ] Timeline or duration provided
