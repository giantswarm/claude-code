## Labels

**Required:**
- `team/<name>` - Owning team
- `kind/recurring`

**Recommended:**
- `function/<area>` - For cross-functional tasks

## Examples

Good recurring titles:
- `Monthly: Review and update PagerDuty on-call schedules`
- `Quarterly: Security audit of cluster access permissions`
- `Weekly: Triage open support tickets and assign owners`

## Quality Checklist

- [ ] Clear task description
- [ ] Frequency specified
- [ ] Owner identified
- [ ] Checklist of steps provided
- [ ] Purpose/context explained
