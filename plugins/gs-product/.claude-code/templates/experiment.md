## Labels

**Required:**
- `team/<name>` - Experiment owner
- `kind/experiment`

**Recommended:**
- `working-group/hackathon` - If from hackathon
- `working-group/ai` - If AI-related experiment
- `label:POC` - For proof of concepts

## Examples

Good experiment titles:
- `Experiment: Validate KubeEdge for edge device management`
- `POC: Test Cilium network policies for multi-tenant isolation`
- `Hackathon: Explore AI-powered incident root cause analysis`
