# gs-ae

Commands for Account Engineers

## Version

0.1.0

## Available Commands

### `/gs-ae:customer-update`

Generate an overview over issues for a given customer repository over a configurable timeframe.

## Installation

See the main [README.md](../../README.md) for general installation instructions.

## Requirements

- GitHub MCP tools (use `/gs-base:init-github` to set up)
- Read-only access to Github issues (and metadata) is sufficient
